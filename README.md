# Protocol for Efficient Navigation and Intercommunication Sequences (PENIS)
PENIS is a networking protocol designed for communication between a wheeled ball-collecting robot and a controller, which is connected to a camera. This repository is a part of the project completed in the course 62410 CDIO at DTU. This repository contains both the protocol specification and an implementation for use in the controller and robot implementations.
